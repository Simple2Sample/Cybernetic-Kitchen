I denne oppgaven ser vi på problemet med å finne korteste vei alle-til-alle i en rettet graf. Du skal benytte deg av følgende algoritme, beskrevet i forelesning: 

SLOW-ALL-PAIRS-SHORTEST_PATH(W)
n = W.rows
L(1) = W
for m = 2 to n-1
    let L(m) be a new (n x n) matrix
    L(m) = EXTEND-SHORTEST-PATHS(L(m-1), W)
return L(n-1)

EXTEND-SHORTEST-PATHS(L, W)
n = L.rows
let L' = (l'ij) be a new (n x n) matrix
for i = 1 to n
    for j = 1 to n
        l'ij = Infinity
        for k = 1 to n
            l'ij = min(l'ij, lik + wkj)
return L'

Oppgave 3

Warshalls algoritme for å finne transitiv tillukning av en graf er som følger: 

WARSHALL(G)
1. n = |G.V|
2. let T(0) = (tij(0)) be a new (n x n) matrix
3. for i = 1 to n
4.     for j = 1 to n
5.         if (i, j) is in G.E
6.             tij(0) = 1
7.         else tij(0) = 0
8. for k = 1 to n
9.     let T(k) = (tij(k)) be a new (n x n) matrix
10.    for i = 1 to n
11.        for j = 1 to n
               // | is binary OR, & is binary AND
12.            tij(k) = tij(k-1) | (tik(k-1) & tkj(k-1))
13.return T(n)
